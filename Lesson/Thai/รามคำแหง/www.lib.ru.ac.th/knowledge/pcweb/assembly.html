หามาเพิ่มเติม ต่อจากของอาจารย์พรชัยครับ

  MASM Assembler ของ microsoft มีให้ฟรีอยู่แล้ว ในแผ่น Win98 ถ้าไม่มี ก็ให้ download file แต่ละตัวมาติดตั้งก่อน ที่นำมาให้ Download เป็น Version 6.11d ได้แก่ 
  • BINS_DDK.EXE (2.5 MB.) 
  • 98SETUP.EXE (3.4 MB.)
  • Lnk563.EXE (275 KB.)  

        สำหรับ Link563.EXE  ที่ต้องติดตั้งลงไปนั้น เพราะว่า LINKER ที่มากับ DDK จะไม่ process 16-bit application เลยต้องลงตามไป 
        และที่ไม่ใช้เป็น Version 6.11c นั้น เพราะ 6.11c เป็น network ทาง microsoft บอกว่า จะไม่สามารถ patch เป็น Version 6.14 ได้

  จากนั้นก็ติดตั้งลงไปบน HDD ของเราซะเลย โดย

  • สร้าง New Folder เป็น Temp ไว้ก่อน 1 อัน
  • Run  BINS_DDK.EXE แล้วระบุ ให้แตกไปที่ Folder ที่สร้างไว้  
  • Run  98SETUP.EXE แล้วระบุ ให้แตกไปที่ Folder ที่สร้างไว้ again... 
  • มองหา file ชื่อ SETUP.EXE แล้วก็ Run มันซะ เราระบุ path เองใหม่ได้ เช่น D:\98DDK  โปรแกรมบอกว่าจะใช้เนื้อที่ประมาณ 6 MB. 
  • reboot 1 ครั้ง ตาม microsoft style 
  • copy Link563.EXE ไปไว้ใน Folder ที่เราติดตั้งไปแล้ว จะเห็นว่า มี file ชื่อ link.exe อยู่แล้ว ตัวที่เห็นจะเป็นตัวเก่า Version 5.12 
  • Run Link563.EXE แล้วให้ทับของเก่าไปเลย เป็นอันเสร็จพิธี ขั้นที่ 1 

  ขั้นตอนต่อไป patch ให้เป็น Version 6.14 download มาก่อนนะ ML614.EXE (859 KB.) 

  • copy file ML614.EXE ไปไว้ใน folder BIN 
  • Run มันเลย จะได้ file เพิ่มมาอีกนิดหน่อย ถือว่าเรียบร้อย...
