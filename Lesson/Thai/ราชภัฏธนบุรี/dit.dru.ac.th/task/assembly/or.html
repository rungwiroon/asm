คำอธิบายคำสั่งทางตรรกของ OR

การลบด้วยตัวมันเองก็จะเซตค่าเป็นศูนย์ได้เหมือนกัน ยังมีคำสั่งที่เป็นญาติกับเอ็กซ์คลูซีฟออร์ คือ ออร์ (OR) อยู่อีกที่เราสามารถนำมาใช้แทนคำสั่งบางคำสั่งได้

การตรวจข้อมูลในรีจิสเตอร์ AX ว่าเป็นศูนย์หรือไม่ เราอาจใช้คำสั่ง CMP AX,0 แต่เราสามารถใช้คำสั่งแทนที่มีประสิทธิภาพกว่าคือการใช้คำสั่ง OR,AX,AX เพราะว่า ออร์ (OR) มีตารางความจริงเป็นดังนี้

OR 0 1

0

 0 1

1

 1 1

จะเห็นได้ว่าผลที่ได้จะเป็นศูนย์ก็ต่อเมื่อข้อมูลเป็นศูนย์ นอกนั้นเป็นหนึ่ง (จริง) หมด เพราะฉะนั้นจากคำสั่ง OR,AX,AX รีจิสเตอร์ AX มีค่าเท่าเดอม แต่จะมีผลต่อแฟลก Z เท่านั้น

นอกจากการใช้คำสั่ง OR ในการเปรียบเทียบค่าแล้ว เรายังสามารถใช้ในการเซตบิตที่ต้องการให้เป็นหนึ่งได้อีกด้วยเช่น ต้องการเซตบิตที่สามให้เป็นหนึ่งก็สั่งได้ดังนี้

    1011   0101
OR   0000   0000
    -------- --- --------
    1011   0101

สมมติให้ AX มีข้อมูล 1011 0101 หลังจากการ OR AX,0000 100b แล้วจะทำให้บิตที่สาม AX เป็นหนึ่งเสมอ

จำนวน clock : (80286)2-7,(80386)2-7

แฟลกที่ถูกกระทำ : OF=0,CF=0,PF,SF,ZF

แฟลกที่ไม่ต้องกำหนด : AF

ข้อยกเว้นในโหมด protected : ถ้าผลลัพธ์ที่ได้อยู่ในเซกเมนต์ที่เขียนข้อมูลได้ ยกเว้นการผิดพลาดนั้นเกิดขึ้น ถ้าใช้เซกเมนต์ CS,DS หรือ ES เก็บค่าหน่วยความจำของโอเปอร์แรนด์

ข้อยกเว้นในโหมด real address : เมื่อโอเปอร์แรนด์อยู่ที่ตำแหน่งออฟเซ็ต OFFFH ทำให้เกิด int 13

80386 note : ใช้กับเวิร์ดขนาด v1K

สำหรับการทำให้เข้าใจโปรแกรมขึ้นนั้นข้อมูลควรจะอยู่ในรูปของเลขฐานสอง ซึ่งจะเก็บขนาด 1 ไบต์ (DB ถ้ามีการใช้ซอฟแวร์ดีบัก เซกเมนต์ข้อมูลจะปรากฎให้เห็นรูปของเลขฐานสิบหก ส่วนโปรแกรมสำหรับการจำลอง วงจรนั้นมีตัวอย่างดังนี้คือ

;FOR 8088/80386 MACHINES

;PROGRAM TO ILLUSTRATE THE LOGIC OPERATIONS ;AND/OR/XOR/NOT

STACK SEGMENT PARA STACK

DB 64 DUP(MYSTACK)

STACK ENDS

MYDATA SEGMENT PARA DATA

IN1 DB 10111010B

IN2 DB 11111100B

IN3 DB 00010100B

IN4 DB 00010011B

IN5 DB 7

MYDATA ENDS

MYCODE SEGMENT PARA CODE ;DEFINE CODE SEG OFR MASM

MYPROC PROC FAR ;PROCEDURE IS NAMES MYPROC

ASSUME CS:MYCODE,DS ;MYDATA,SS:STACK

PUSH DS ;SAVE LOCATION OF DS REG.

SUB AX,AX ;GET A ZERO IN AX

PUSH AX ;SAVE ZERO ON STACK, TOO

MOV AX,MYDATA ;GET DATA LOCATION IN AX

MOV DA,AX ;PUT IT IN DA REGISTER

;EXAMPLE OF LOGICAL GATE SYNTHESIS

MOV AL,IN1 ;LOAD FIRST INPUT INTO AL

NOT AL ;INVERT BITS IN AL

MOV AH,IN2 ;LOAD SECOND INPUT INTO AH

OR AL,AH ;ORTHE TWO

MOV BL,IN3 ;LOAD THIRD INPUT INTO BL

AND BL,IN4 ;AND THE TWO

XOR AL,BL ;EX-OR OUTPUTS OF GATES

MOV ANS,AL ;SAVE RESULT IN ANS(WER)

;COMPLETION OF GETE SYNTHESIS EXAMPLE

RET ;RETURN CONTROL TO DOS

MYPROC ENDP ;END PROCEDURE NAMED MYPROC

MYCODE ENDS ;END CODE SEGMENT NAMED

;MYCODE

END ;END WHOLE PROGRAM
