2. DTA = DATA TRANSFER AREA หมายถึง เนื้อที่ของข้อความที่ควบคุมได้ หรือตำแหน่งของ MEMORY

ที่เก็บข้อมูลเข้าหรืออ่านข้อมูลออก คือ เป็นพื้นที่ MEMORY ที่ทำหน้าที่ถ่ายเทข้อมูลเข้าออก โดย 128 BYTE = 1 BLOCK

 

DOS FILE STRUCTURE คือ จาก FILE แยกเป็น BLOCK และจาก BLOCK แยกเป็น RECORD

DISK-ACCESS METHODS

DISK-ACCESS METHODS คือ หลักการเขียนโปรแกรมในการเข้าถึง DISK

1. SEQUENTAL

1. เปิดหรือสร้างไฟล์

2. CLEAR ค่าRECORD-SIZE FIELD เป็นค่าที่ต้องการ

3. CLEAR ค่า CURRENT-BLOCK และ CURRENT-RECORD FIELD เป็นศูนย์

4. ไปทำการใน FUNCTION อ่านหรือบันทึก แบบ SEQUENTAL

5. ทำ ข้อ 4 จนกระทั่ง จัดการ FILE สมบูรณ์

6. ปิดไฟล์

2. RANDOM

1. เปิดหรือสร้างไฟล์

2. CLEAR ค่า RECORD-SIZE FIELD เป็นค่าที่ต้องการ

3. กำหนด RELATIVE-RECORD FIELD เพื่อชี้ RECORD ที่ต้องการ

4. ไปทำการใน FUNCTION อ่านหรือบันทึก แบบ RANDOM

5. ทำ ข้อ 3 และ4 จนกระทั่ง จัดการ FILE สมบูรณ์

6. ปิดไฟล์

3. RANDOM BLOCK

1.เปิดหรือสร้างไฟล์

2. CLEAR ค่า RECORD-SIZE FIELD เป็นค่าที่ต้องการ

3. กำหนด RELATIVE-RECORD FIELD เพื่อชี้ RECORDที่ต้องการ

4. กำหนด CX= จำนวนของเรคอร์ดที่ต้องการ

5. ไปทำการใน FUNCTION อ่านหรือบันทึก แบบ RANDOM BLOCK

6. ทำ ข้อ 3,4 และ5 จนกระทั่ง จัดการ FILE สมบูรณ์

7. ปิดไฟล์
