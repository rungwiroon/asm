Addressing Modes.

วิธีการอ้าง Address ( Addressing Modes )

Addressing Modes เป็นวิธีการที่ไมโครโปรเซสเซอร์จะเข้าถึงออปเปอแรนด์ ที่โปรแกรมต้องการทำปฏิกิริยาอย่างใดอย่างหนึ่งขึ้นอยู่กับคำสั่งออปเปอแรนด์อาจเป็นส่วนหนึ่งของคำสั่งงงอยู่ในรีจิสเตอร์อยู่ในหน่วยความจำสำหรับไมโครโปรเซสเซอร์ 8088 มีวิธีการแบ่ง Addressing Modes เป็นดังนี้

      Immediate addressing
      Register addressing
      Direct addressing
      Register indirect addressing
      Base relative addressing
      Direct indexed addressing
      Base indexed addressing

 

Immediate Addressing Mode

เป็นการให้ค่าคงที่ซึ่งอาจเป็น 8 บิต หรือ 16 บิต กับออปเปอแรนด์ โดยให้นิยามค่าคงที่นั้นตามหลังคำสั่ง ( op-code ) ค่าบวกลบของเลข 8 บิต มีค่าอยู่ระหว่าง -128 ( 80H ) ถึง 127 ( 7FH ) และค่าบวกลบสำหรับเลข 16 บิตมีค่าอยู่ระหว่าง -32768 ( 8000H ) ถึง 32767 ( 7FFFH ) ค่าที่ไม่มีเครื่องหมายของเลข 8 บิต และ 16 บิต มีค่าต่ำสุดคือ 0 ค่า สูงสุดสำหรับ 8 บิต คือ 255 ( 0FFH ) และค่าสูงสุดสำหรับ 16 บิต คือ 65535 ( 0FFFFH )

Register Addressing Mode

8088 จะนำค่าของออปเปอแรนด์มาจากรีจิสเตอร์ซึ่งขึ้นอยู่กับรีจิสเตอร์ที่ใช้ว่าเป็น 8 บิต หรือ 16 บิต เช่น

1. MOV AX,CX

2. INC AX

3. ADD SI,CX

4. DEC DX

Direct Addressing Mode

สำหรับ direct addressing mode ค่าของ EA จะตามหลัง op-code 8088 จะหาค่าของออปเปอแรนด์ โดยการบอกค่าของ EA กับค่าที่อยู่ในดาตาเซกเมนต์รีจิสเตอร์ ออปเปอแรนด์อาจจเป็นเลเบลแทนที่จะเป็นค่าออฟเซต

Register Addressing Mode

สำหรับ register indirect addressing ค่าของ EA ของออปเปอแรนด์จะอยู่ในเบสรีจิสเตอร์ BX, เบสพอยเตอร์ BP หรืออินเด็กซ์รีจิสเตอร์ในการเขียน register indirect addressing mode ต้องใช้รีจิสเตอร์ภายในวงเล็บ

ฺBase Relative Addressing Mode

สำหรับ base relative addressing mode ค่าของ EA ของออปเปอแรนด์คือค่าผลบวกของออฟเซตและค่าที่อยู่ในเบสรีจิสเตอร์ BX หรือเบสพอยเตอร์ BP การใช้เบสรีจิสเตอร์นั้นให้ความสะดวกในการเข้าถึงข้อมูลซึ่งอยู่ ณ ตำแหน่งต่างกันในหน่วยความจำ โดยการนำค่าของเบสแอดเดรสของข้อมูลใส่ไว้ในเบสรีจิสเตอร์และอ้างถึงข้อมูลโดยใช้ค่าระยะทางที่ห่างจากเบสแอดเดรสหรือค่าออฟเซตนั่นเอง

Direct Indexed Addressing Mode

สำหรับ direct indexed addressing mode ค่า EA ของออปเปอแรนด์ คือผลบวกของออฟเซตและค่าที่อยู่ในอินเด็กซ์รีจิสเตอร์ DI หรือ SI ก็ได้โหมดนี้นิยมใช้กับการประมวลผลข้อมูลกับตารางซึ่งออฟเซตจะชี้ไปที่ค่าซึ่งอยู่ภายในตารางนั้น

Base Indexed Addressing Mode

สำหรับ base indexed addressing mode Ea ของออปเปอแรนด์ คือผลบวกของค่าออฟเซตค่าของเบสรีจิสเตอร์และค่าของอินเด็กซ์รีจิสเตอร์ ค่าของออฟเซตอาจมีหรือไม่มีก็ได้ base indexed addressing mode นิยมใช้ในการประมวลผลข้อมูลที่เกี่ยวกับตาราง 2 มิติ ซึ่งเบสรีจิสเตอร์จะเก็บค่าเริ่มต้นของตารางค่าของออฟเซตจะชี้บอกแถวนอนของตารางและค่าของอินเด็กซ์รีจิสเตอร์จะชี้บอกแถวตั้งของตาราง
