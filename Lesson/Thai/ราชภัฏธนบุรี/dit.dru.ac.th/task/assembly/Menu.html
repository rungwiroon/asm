ยินดีต้อนรับเข้าสู่บทเรียนภาษาแอสเซมบลี เลือกบทเรียนที่คุณต้องการศึกษา                
        ภาษาแอสเซมบลี               คำสั่ง SHL และ SHR
        การใช้ Debug เขียนโปรแกรม               คำสั่ง MUL และ IMUL
        วิธีการอ้าง Address ( Addressing Mode )               คำสั่ง DIV และ IDIV
        Macro Assembler               คำสั่ง AND
        Interfacing               คำสั่ง XOR
        Disk I/O               คำสั่ง OR
        ADD ( Addition )               ชิป : ก้าวใหม่เทคโนโลยีคอมพิวเตอร์
        คำสั่ง SUB และ SBB                  
                             
        ตัวอย่างโปรแกรม
