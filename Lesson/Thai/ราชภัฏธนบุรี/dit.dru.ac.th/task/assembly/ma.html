MACOR ASSEMBLER
 

ASSEMBLER คือตัวแปลภาษา NEMONIC CODE เป็นภาษาเครื่องในASSEMBLER จะมี DATA TYPE มีจุด DB เป็น DATA TYPE

การกำหนดข้อมูลในภาษาแอสเซมบลี

DB กำหนดข้อมูลชนิดเป็น BYTE จำนวนไบท์ = 1(8 BITS) สถานะภาพ BYTE DEFINE BYTE

DW กำหนดข้อมูลชนิดเป็น WORD จำนวนไบท์ = 2 สถานะภาพ WORD DEFINE WORD

DD กำหนดข้อมูลชนิดเป็น DOUBLE จำนวนไบท์ = 4 สถานะภาพ DOUBLE DEFINE DOUBLE

Q กำหนดข้อมูลชนิดเป็น QUAD จำนวนไบท์ = 8 สถานะภาพ QUAD DEFINE QUAD

T กำหนดข้อมูลชนิดเป็น TEN จำนวนไบท์ = 10 สถานะภาพ TEN DEFINE TEN

THE MODEL DIRECTIVE.

MOEL DESCRIPTION

TINY โปรแกรมและข้อมูลรวมกันไม่เกิน 64 K

SMALL โปรแกรมหรือไม่ก็ข้อมูลอย่างใดอย่างหนึ่งไม่เกิน 64 K

MEDIUM โปรแกรมเท่านั้นที่เกิน 64 K

COMPACT ข้อมูลเท่านั้นที่เกิน 64 K

LARGE ทั้งโปรแกรมและข้อมูลเกิน 64 K ได้

HUGE MEMORY ที่มีค่าใช้ได้ทั้งหมด ถูกใช้เพื่อ โแรกรมและข้อมูล

THE INT INSTRUCTION.

1. ต้อง DEFINE ชื่อของ FUNCTION และ SERVICE ให้ DOS โดยบอก INT

2. หา ADDRESS ของ INT ใน TABLE

3. ไป ADDRESS ที่ ROM BIOS แล้วทำ Sub program นั้น

4. IRET = INTRUPT RETURN กลับไป Main program ที่คำสั่งถัดไป โดยไปดู Stack ก่อนใน Assembly ตัว Compile จะเก็บ POP Stack Address ของคำสั่งถัดไปก่อนเสอมจึงจะไปที่คำสั่ง Address ที่ตนอยู่เพื่อให้กลับมาถูก Address เรียนว่า Look Ahead Forword.

คำสั่งปฏิบัติการของภาษาแอสเซมบลี้มี 4 ชนิด.

1. ภาษาเครื่อง

2. ตัวแปลภาษา

3.ชุดคำสั่ง

4.ตัวบอก ADDRESS DATA

คำสั่งภาษาแอสเซมบลี้ถุกแบ่งออกเป็น 4 เขต คือ.

LABEL การกระทำ ตัวถูกกระทำ ; COMMENT

ASM ยาวได้ 31 ตัว

BISIC ยาวได้ 8 ตัว

ไมโครโปรเซสเซอร์ 8088 จะเกิดการย้าย 2 ข้อ.

1. ต้องตามด้วย : CODE SEGMENT เดียวกัน CS ไม่เปลี่ยน OFFSET ใส่ใน IP

2. ต่าง CODE SEGMENT กับคำสั่งควบคุม OFFSET ของ LABEL จำใส่ใน IP และ CS

การ GET ASCII โดยใช้ LOOP NEXT.

MOV CX,100H

MOV DL,0

NEXT : MOV AH,02H

INT 21H

ดึงตัวอักษรจาก ASCII CODE.

1. ASCCCII 0-255

2. เลือก Loop โดยใช้รีจีสเตอร์ CX ( COUNTER,LOOP ลด จนเป็น 0 จึงหยุด )

3. เลือกการแสดงผล INT 21h

- FUNCTION 02H

- REGISTER DL

4. TERMINATE PROGRAM โดยใช้ INT 21H

- FUNCTION 4CH
