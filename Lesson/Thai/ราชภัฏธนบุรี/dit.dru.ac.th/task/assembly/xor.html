คำสั่ง XOR

AYNTAX XOR destination,source

คำอธิบายทางตรรก

ทีนี้ลองมาดูว่าทำไมคำสั่ง XOR,AX,AX จึงสามาารถเซตข้อมูลในรีจิสเตอร์เป็นศูนย์ได้ คำสั่งนี้เป็นคำสั่งทางลอจิก อ่านว่า เอ็กซ์คลูซีฟออร์ (Exclusive OR) ซึ่งมีค่าตารางความจริง (true table) ดังนี้

XOR 0 1

0

 0 1

1

 1 1

จะเห็นว่าถ้าบิตใดบิตหนึ่งเป็นจริงเพียงบิตเดียว แต่ไม่จริงทั้งสองบิตพร้อมกัน จะได้ค่าเป็นจริง ดังนั้นถ้า AX มีข้อมูล 1011 0101 อยู่ เมื่อพบคำสั่ง XOR,AX,AX ผลลัพธ์จะเป็นศูนย์เสมอ

    1011   0101
XOR   1011   0101
    ------- -- -------
    0000   0000

จำนวน clock : (80286)2-3,(80386)2-7

แฟลกที่ถูกกระทำ : OF=0,CF=0,PF,SF,ZF

แฟลกที่ไม่ต้องกำหนด : AF

ข้อยกเว้นในโหมด protected : ถ้าผลลัพธ์ที่ได้อยู่ในเซกเมนต์ที่เขียนข้อมูลได้ยกเว้นการผิดพลาดนั้นเกิดขึ้น ถ้าใช้เซกเมนต์ CS,DS หรือ ES เก็บค่าหน่วยความจำของโอเปอร์แรนด์ EA ก็จะเกิดการยกเว้น แต่ถ้าเป็นเซกเมนต์ SS จะเกิดความผิดพลาดที่ stack ขึ้น

ข้อยกเว้นในโหมด real address : เมื่อโอเปอร์แรนด์อยู่ที่ตำแหน่งออฟเซ็ต OFFFH ทำให้เกิด int 13 80386 note : ใช้กับเวิร์ดขนาด 32 บิต

PUSHA MACRO ;SAVES SYSTEMS REGISTER

PUSH AX ;NOT NEEDED BY 80286/80386

PUSH CX ;MACHINES

PUSH DX

PUSH BX

PUSH BP

PUSH SI

PUSH DI

ENDM

POPA MACRO ;RESTORES SYSTEM REGISTER

PUSH DI ;NOT NEEDED BY 80286/80386

PUSH SI ;MACHINES

PUSH BP

PUSH BX

PUSH DX

PUSH CX

PUSH AX

ENDM

DELAY MACRO TIME ;SOFTWARE RIME DELAY CONIROLLED

LOGICAL P1,P2 ;BY (TIME) AND SYSTEM CLOCK TATE

PUSHA ;SAVE REGISTER

MOV DX,TIME ;MOVE LARCE UNITS INTO DX

P1 : MOV CX0F00H ;SMALL INCREMENTS SET IN CX

P2 : DEC CX ;DECREMENT CX

JNZ P2 ;IFNOTZERO,P2

DEC DX ;IF ZERO,DECREMENT DX

JNZ P1 ;IF DX NOT ZERO, P1

PAPA ;RESTORE REGIXTER

ENDM ;END MACRO

CLEARSCREEN MACRE ;CLEARS COLOR SCREEN

PUSHA ;SAVE REGISTER

MOV CX,0 ;SET UPPER CORNER OF WINDOW

MOV DX,2479H ;SET LOWER CORNER OF WINDOW

MOV BH,7 ;SET SCREEN ATTRIBYTE(NORMAL)

MOV AX,0600H ;INTERRUPT PARAMETERS

INT 10H ;CALL INTERRUPT

POPA ;RESTORE TEGISTERS

ENDM ;END MACRO

PRINTNUM MACRO ;PRINTS ASCII NUMBER ON SCREEN

PUSHA ;SAVE REGISTER

PUSH AX ;SAVE AX IT WILL BE DESTROVED

MOV AH,15 ;GET CURRENT SCREEN

INT 10H ;CALL INTERRUPT, SET BX VALUES

POP AX ;RESTORE AX VALUES

AND AL,0FH ;KEEP ONLY LOWER 4 BITS

OR AL,30H ;MAKE IT ASCII

MOV AH,10 ;SET INTERRUPT PARAMETER

MOV CX,1 ;NUMBER OF CHAR, TO WRITE

INT 10H ;CALL THE INTERRUPT

POPA ;RESTORE THE REGISTERS

ENDN ;END MACRO

PRINTCHAR MACRO TEXT ;PRINT CHARACTERS FROM DATA

PUSHA ;SEGMENT, STRING TO BE PASSED

LEA DX,TEXT ;IN (TEXT),PRINTF UNTIL ($)

MOV AH,9 ;IS LOCATED, PRONTS FROM CURRENT

INT 21H ;CURSOR POSITION

POPS ;RESTORE THE REGISTERS

ENDM ;END MACRO
